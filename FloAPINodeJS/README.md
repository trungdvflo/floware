## FloAPI version 4
We use NestJS for new api. We move all api write by ruby on rails to NestJS. 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test report
$ npm run report:serve
```

## Folders

```js
+-- bin // Custom tasks
+-- dist // Source build
+-- node_modules // npm install modules
+-- public // Static Files
+-- migrations
+-- libs 
|   +-- cust-library
|   |   +-- index.ts
|   |   +-- *
+-- src
|   +-- configs // Environment Configuration
|   +-- auths // Authentication
|   +-- commons // Global Nest Module
|   |   +-- caches // Cache adapter
|   |   +-- constants // Constant value and Enum
|   |   +-- databases // Connect database typeorm, import entities
|   |   +-- logs  // Sentry, graylog, system log
|   |   +-- interfaces // TypeScript Interfaces
|   |   +-- middlewares // Nest Middleware
|   |   +-- exceptions // Exception error for Nest @catch
|   |   +-- * // models, repositories, services...
|   +-- {{module_name}} // Other Nest Modules, non-global
|   |   +-- dtos // DTO (Data Transfer Object) Schema, Validation
|   |   +-- entitys // TypeORM Entities generated by `typeorm-model-generator` module
|   |   +-- helpers // Help for controller 
|   |   +-- tests // Jest testing
|   |   +-- utils
|   |   +--{{module_name}}.controller.ts // Nest generator 
|   |   +--{{module_name}}.module.ts    //  Nest generator 
|   |   +--{{module_name}}.service.ts   //  Nest generator 
|   |   +--{{module_name}}.providers.ts  // Custom provider if need
```
